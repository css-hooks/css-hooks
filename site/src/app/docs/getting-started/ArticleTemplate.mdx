## Installation

Run the following command to install CSS Hooks for {props.framework}:

<Pre>
  <Code>npm install {props.package}</Code>
</Pre>

This will also install the `hooks.css` package, used in the next step,
automatically.

## CSS import

Import `node_modules/hooks.css/index.css` into your app. Depending on your
bundler configuration, you may be able to import `hooks.css` as a shorthand for
that file path.

{props.cssImportExample}

## Use hooks!

First, import the `hooks` helper function from the <Code>{props.package}</Code>
module.

Then, determine the element where you want to use hooks. Simply add the `hooks`
utility class; then, call the `hooks` function with your inline styles, passing
the result to the `style` prop as usual. For example:

{props.hookUsageExample}
